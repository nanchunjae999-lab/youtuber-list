키워드로 채널을 찾고, 조건으로 거르고, 이메일까지 한 번에

API 키 설정

YouTube Data API v3 키를 한 줄에 하나씩 입력하세요. 여러 개 입력 시 쿼터 소진되면 자동으로 다음 키로 전환됩니다. (브라우저에만 저장됩니다)

아래 순서대로 버튼 이름 그대로 따라 누르기만 하면 됩니다. 5분이면 끝나요!

  1. 구글 클라우드 접속
    여기를 클릭해서 구글 계정으로 로그인하세요. 처음이면 "서비스 약관" 체크박스 체크 → [동의 및 계속하기]를 누릅니다.
  2. 프로젝트 만들기
    화면 왼쪽 위에 있는 프로젝트 선택 버튼(▼) 클릭 → 창이 뜨면 오른쪽 위 [새 프로젝트] 클릭 → 이름은 아무거나 (예: my-youtube) → [만들기] → 잠시 기다렸다가 다시 프로젝트 선택 버튼(▼)을 눌러 방금 만든 프로젝트를 클릭해서 선택합니다.
  3. 유튜브 기능 켜기
    여기를 클릭 → 파란색 [사용] 버튼을 누릅니다.
    ⚠️ 이 단계를 빼먹으면 키를 만들어도 작동하지 않아요!
  4. API 키 만들기
    여기를 클릭 → 위쪽의 [+ 사용자 인증 정보 만들기] 클릭 → 메뉴에서 [API 키]를 선택합니다.
  5. "어떤 API를 사용하나요?" 라고 물어보면
    API 선택: YouTube Data API v3 를 선택하고, "어떤 데이터에 액세스하나요?" 가 나오면 공개 데이터를 선택 → [다음]을 누릅니다. (이 화면이 안 나오면 그냥 다음 단계로!)
  6. 키 복사하기
    화면에 AIzaSy로 시작하는 긴 글자가 나옵니다. 복사 버튼을 눌러 복사한 뒤, 이 페이지 맨 위의 API 키 칸에 붙여넣으면 끝!
    ⚠️ "OAuth 클라이언트 ID"를 만들면 안 돼요! client_secret 같은 게 나왔다면 잘못 만든 것 → 지우고 4번부터 다시 하세요.
  7. (선택) 키 여러 개 만들기
    하루 사용량(10,000유닛)을 늘리고 싶으면 2번부터 다시 반복해서 프로젝트를 하나 더 만들면 됩니다. 키는 위 칸에 한 줄에 하나씩 넣으면 자동으로 번갈아 사용돼요.

수집 조건

페이지당 쿼터 100유닛 사용. 검색된 영상의 채널을 후보로 수집합니다.

~

비워두면 해당 방향 제한 없음. 구독자수 숨김 채널은 제외됩니다.

% 이상

공식: 최근 영상 평균 조회수 ÷ 구독자수 × 100  |  30% 이상이면 준수, 100% 이상이면 떡상 채널

% 이상

공식: (좋아요 + 댓글) ÷ 조회수 × 100  |  평균 1.4% · 2%↑ 좋음 · 5%↑ 최상위

개 이상
개 이상

채널의 최근 영상 평균 기준. 비워둔 항목은 적용되지 않습니다.

채널의 평균 조회수·참여율을 계산할 때 최근 영상 몇 개를 표본으로 쓸지 정하는 옵션입니다.
예: "최근 10개"로 설정하면 → 각 채널의 최신 영상 10개를 가져와서 평균 조회수, 참여율(좋아요+댓글÷조회수), 평균 댓글·좋아요를 계산 → 그 값으로 2~4번 필터를 적용합니다.
적을수록 채널의 최신 상태를 민감하게 반영, 많을수록 안정적인 평균이 됩니다. 보통 10개면 충분합니다.
